# Rest API Client (Flutter)

A Flutter application that demonstrates best practices for consuming REST APIs using clean architecture and MVVM.

## Features

* Fetch data from REST APIs
* Display lists and details
* Error handling
* Loading states
* Pagination (coming soon)

## Tech Stack

* Flutter
* Dio (Networking)
* Provider (State Management)
* Freezed (Data Models)

## Architecture

* MVVM + Repository Pattern

## Getting Started

```bash
flutter pub get
flutter run
```
